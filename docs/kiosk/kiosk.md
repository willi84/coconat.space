# 🍓 Raspberry Pi Kiosk Setup

Welcome to the **Raspberry Pi Kiosk Setup** guide!  
This project turns a Raspberry Pi into a sleek, headless **kiosk system** — perfect for digital signage, dashboards, or info screens.

With this setup, your Raspberry Pi will:

✅ Boot directly into a fullscreen Chromium browser  
✅ Auto-connect to Wi-Fi  
✅ Display a custom website with no need for keyboard or mouse

Whether you're setting it up for the first time or just need a quick restart, we've got you covered.
